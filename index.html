Why am I in web design?

I am in web design this year because last year when I was in Multimedia I the web design portion of the class was my favorite part of the year. It was interesting and new and I thought it was really fun too.

How is knowing HTML code helpful?

Knowing HTML is beneficial because the internet is becoming more prevalent with business and technology. So having an advantage over other people when it comes to understanding the code is useful. This can be very important if your job has to do with websites or running websites.

What might be a future career path?

There are many career paths when it comes to knowing these skills; you could be a web designer for companies, you can manage people's sites, or run a site of your own. A career path that I would love to do would be a web designer. There are potentially hundreds of jobs that this knowledge can be used to help with.
